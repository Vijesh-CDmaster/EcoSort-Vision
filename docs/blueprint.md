# **App Name**: EcoSort Vision

## Core Features:

- Dashboard: Real-time overview of waste levels, segregation performance, and recycling rates.
- Waste Detection: Utilize camera to identify the waste type.
- Segregation Decision: Employ a image-classification tool to determine the appropriate waste bin for each item.
- Contamination Alerts: Issue immediate notifications upon detecting incorrect waste disposal.
- Bin Monitoring: Keep track of waste bin fullness levels to optimize collection schedules and routes.

## Style Guidelines:

- Primary color: Black (#000000) for focus.
- Background color: White (#FFFFFF), providing a clean and neutral base.
- Accent color: Light gray (#D3D3D3) to subtly highlight important information.
- Body and headline font: 'PT Sans', a clean and readable sans-serif, similar to the fonts used by Notion. The user wants the frontend to resemble Notion.
- Simple, minimalist icons in a style consistent with Notion, to enhance usability.
- Adhere to a clean, grid-based layout inspired by Notion, for easy content organization and navigation.
- Subtle, smooth transitions and animations to provide a fluid and engaging user experience.
